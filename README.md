# 🚲 Bikenolimit

**Bikenolimit** este o platformă web dedicată performanței pe două roți, servind drept magazin online și service specializat pentru cicliști. Site-ul oferă o structură completă pentru comercializarea de biciclete (MTB, Șosea, Electrice), piese de schimb și echipamente premium. 

Proiectul pune bazele unei prezențe online profesionale prin:
* **Catalog de Produse:** Categorisire precisă a bicicletelor și componentelor.
* **Centru de Service:** Listă detaliată de tarife și pachete de revizie pentru întreținere tehnică.
* **Comunitate:** Secțiuni dedicate evenimentelor și tutorialelor video de mentenanță.

Implementat ca etapă a disciplinei **Tehnologii Web**, proiectul utilizează standarde HTML5 semantice pentru a asigura accesibilitatea și o indexare SEO optimă.
